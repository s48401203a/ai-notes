1. 工具概述

本工具用于每周处理电脑管理平台导出的数据文件，自动筛选华南区域相关数据，生成可视化HTML周报。

主要功能

  • 自动识别3类数据文件（未安装火绒、未编辑、可利旧）
  • 按区域筛选数据（华南区域 + 指定省份城市）
  • MAC地址去重
  • 生成可折叠的HTML报告
  • 周数据汇总与环比分析
  • 历史数据保留

2. 文件结构

文件名 类型 说明
weekly_report.py 主脚本 数据处理和报告生成的核心脚本
weekly_report.html 输出文件 生成的HTML周报，包含历史数据
weekly_history.json 历史数据 存储每周统计数据，用于环比计算
负责的省份和城市列表.txt 配置文件 华南区域省份和城市关键词列表
technical_doc.html 文档 本技术文档

3. 数据源要求

工具会自动扫描工作目录，根据文件名规则识别数据文件：

数据类型 文件名规则 格式
未安装火绒终端 文件名包含 unmanaged_devices xlsx
未编辑终端 文件名包含 edit_status_report csv
可利旧终端 文件名包含 use_status_report csv
注意：每种类型只会识别一个文件。如有多个同类文件，请保留最新的一个。

去重规则按MAC地址去重

6. 使用方法

6.1 运行前准备

  1. 将3类数据文件放入工作目录 D:\\data\\powerquery\\
  2. 确保文件名符合识别规则
  3. 确保 Python 环境已安装依赖库

6.2 依赖库

pip install pandas openpyxl

6.3 运行命令

python D:\\data\\powerquery\\weekly_report.py

6.4 运行输出

脚本会显示处理进度：

  • [1/6] 解析城市列表
  • [2/6] 加载历史数据
  • [3/6] 识别数据文件
  • [4/6] 处理未安装火绒数据
  • [5/6] 处理未编辑和可利旧数据
  • [6/6] 生成HTML报告
注意：每次运行会在报告顶部追加新的周数据。如需重新生成当周数据，请先删除 weekly_report.html 文件。
